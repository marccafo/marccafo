## 🚀 What I'm working on

### ⚡ [Photonne](https://github.com/marccafo/photonne) — *Self-hosted · Every Photon*
A self-hosted photo & video management app (think Immich, but built on .NET).
- 🖥️ **Backend:** ASP.NET Core · **Frontend:** Blazor WASM · **Mobile:** MAUI · **Deploy:** Docker Compose
- 📦 Open source under **AGPL v3** — free core, sustainable open-core model
- 🎯 MVP in production, PWA in progress, native mobile on the horizon
